Signatur Control

Unterschriften-PAD direkt im SAP-Prozess

Signatur Control bindet Signaturpads direkt an SAPGUI an. Anwender unterschreiben in der SAP-Anwendung, das Control exportiert das Signaturbild in verschiedenen Formaten und stellt es für Formulare, PDF-Anhänge und Archivierung bereit.

Signaturprozess

Elektronische Signaturen ohne Medienbruch in SAP übernehmen

Das Control ermöglicht eine elektronische Signatur direkt in der SAP-Anwendung. Das Signaturbild wird an SAP übergeben und kann in verschiedenen Formaten für Folgeprozesse genutzt werden.

Ein direkter Ausdruck auf Formulare, zum Beispiel Lieferscheine, ist ebenso möglich wie die Ablage eines signierten PDF-Anhangs im SAP-Archiv. Damit bleiben Übergabe-, Liefer- und Freigabeprozesse vollständig dokumentiert.

Ablauf

Vom SAP-Vorgang zur archivierten Unterschrift

01

Vorgang starten

Die Signatur wird direkt aus einem SAPGUI-Vorgang heraus angefordert.

02

Pad ansteuern

Das angebundene Signaturpad erfasst die Unterschrift des Anwenders.

03

Signatur übergeben

Das Signaturbild wird exportiert und für Formular oder PDF bereitgestellt.

04

Archivieren

Der signierte Beleg wird im SAP-Prozess verbucht und nachvollziehbar abgelegt.
